# IDP - IA
1.2026 - Inteligência Artificial

Este repositório armazena os códigos, scripts e exercícios desenvolvidos durante as aulas de Python no IDP.

Descrição
O objetivo deste espaço é registrar o aprendizado da linguagem, servindo como consulta para a sintaxe e bibliotecas exploradas. O conteúdo abrange desde conceitos básicos de lógica de programação até o uso de bibliotecas específicas para manipulação de dados.

Estrutura do Repositório
O conteúdo está organizado por temas ou listas de exercícios:
/
├── Lógica_Básica/
├── Listas_e_Dicionarios/
├── Manipulação_Arquivos/
├── Pandas_Introdução/
... e assim por diante

Como Utilizar
Para acessar os arquivos, você pode:

Clonar o repositório

Baixar como ZIP: No GitHub, clique em "Code" e depois em "Download ZIP".

Ambiente e Dependências
A maioria dos scripts foi desenvolvida para rodar em ambientes Python 3.x. Recomenda-se o uso do VS Code ou PyCharm, além da instalação do gerenciador de pacotes pip para eventuais bibliotecas externas.

Contribuições
